  Передбачено два варіанта використання ресурсу:
1. Збір та вигрузка даних про курси з сайту;
2. Користувацький інтерфейс пошуку та відображення курсів;

  Скрипт main.js використовується в стороньому плагіні браузера: "User JavaScript and CSS".
Плагін необхідно попередньо закачати, увімкнути та перейти за адресою сайту: "https://edu.genius.space/uk/courses".
Скрипти js мають бути скопійовані та вставленні у відповідне поле плагіну.

  Для вигрузки данних про курси знадобиться реєстрація та авторизація на сайті (знадобиться 
декілька акаунтів, але по факту достатньо одного email (Gmail) з варіативністю додавання крапок в адресу).

  При жорсткому перезавантаженні сторінки з доступними для ознайомлення курсами на сайті,
скрипт проходиться по курсам, збирає данні з кожної лекції і в подальшому будує деревовидну 
структуру курсів в форматі .json

  Після закінчення роботи надається посилання для вивантаження файлу, який в подальшому має використовуватися 
для перегляду цих курсів (коли доступ до курсів буде скасовано)

  Файл: courses-2023-all.json з курсами агрегується вручну після викачки доступних курсів за існуючою структурою.
  Файл: index.html містить розмітку для відображення користувацького інтерфейсу.
  Файл: grabber.js включає функціональну частину коду.
  Файл: main.js це апаратний скрипт, який має бути завантажений в плагін.

  Файли grabber.js, courses-2023-all.json, та index.html підвантажуються динамічно на сторінку сайту головним
скриптом main.js, який запускається з плагіна.

  Користувацький файл readmeUser.txt має також бути завантажений в гіт для ознайомлення з керівництвом по діям.
